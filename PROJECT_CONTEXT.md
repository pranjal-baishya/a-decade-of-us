# PROJECT: A Decade of Us

## Core Idea
A cinematic, emotionally immersive anniversary web app celebrating a 10-year relationship.

The app should feel like:
- a private memory museum
- an interactive love letter
- a cinematic relationship documentary

NOT:
- a generic portfolio
- a cheesy Valentine template
- a flashy over-animated website

---

# Emotional Direction

The emotional theme is:
“10 years of choosing each other through every version of life.”

The experience should feel:
- warm
- nostalgic
- intimate
- cinematic
- premium
- mature
- emotionally grounded

The app should emotionally progress through:
1. Beginning
2. Growing together
3. Difficult years
4. Choosing each other
5. Future together

---

# Main App Name

A Decade of Us

---

# Tagline

“And somehow, we’re still just getting started.”

---

# Technical Stack

Frontend:
- React
- TypeScript
- Vite
- Tailwind CSS

Animation:
- Framer Motion
- GSAP
- Lenis smooth scroll

3D:
- Three.js
- React Three Fiber
- Drei

Deployment:
- Vercel

---

# Mobile-First Rules

Primary target:
- phones
- tablets

The app MUST:
- work beautifully on mobile
- support portrait orientation
- avoid heavy GPU load
- avoid large particle systems
- keep animations smooth
- maintain readable typography
- support touch interactions

---

# Sections

1. Intro cinematic hero
2. Timeline of 10 years
3. Memory gallery
4. Letter / voice note section
5. Future memories section
6. Final reveal / gift section

---

# Quality Bar

The app should feel:
- emotionally meaningful
- elegant
- highly polished
- cinematic
- intentional

The experience should feel like:
Netflix intro + premium memory journal + emotional documentary.

---

# Design Constraints

Avoid:
- excessive hearts
- childish romance aesthetics
- loud colors
- excessive animations
- cluttered UI

Prefer:
- soft glow
- cinematic lighting
- minimalism
- elegant typography
- glassmorphism
- warm gradients
- subtle stars/particles

---

# Performance Constraints

Optimize for:
- mobile rendering
- smooth scroll
- lazy loading
- code splitting
- reduced motion support

3D should gracefully degrade on weaker devices.
